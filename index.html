Download the boilerplate.

A ready-to-drop folder: the CLAUDE.md contract, the quality-checkpoint skill, and a starter .claude/settings.local.json wired with desktop notifications for when Claude needs you or finishes a task. Pick your operating system — the only difference is how those notifications fire.

macOS

For Mac

Notifications use the built-in osascript with system sounds (Glass / Ping).

Windows

For Windows

Notifications play system sounds via powershell (Exclamation / Asterisk).

Unzip it, then copy CLAUDE.md and the .claude folder into the root of your project. That's it — open the project in Claude Code and the contract, skill, and notifications are live.
